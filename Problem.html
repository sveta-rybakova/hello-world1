Устройство ПК 

 
		     Основных частей у компьютера всего четыре:
		Клавиатура -
		        устройство для ввода информации.
		Память     -
		        устройство для хранения информации.
		Процессор  -
		        устройство для обработки информации.
		Монитор    -
		        устройство для вывода информации.
